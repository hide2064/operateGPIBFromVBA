# CLAUDE.md — プロジェクト規約

## ファイルエンコーディング

| 対象ファイル | エンコーディング | 理由 |
|------------|----------------|------|
| `*.bas` (VBA モジュール) | **SHIFT-JIS** | Excel VBA エディタが SHIFT-JIS を前提とするため |
| `*.py` (Python スクリプト) | **UTF-8** | Python 3 のデフォルト。日本語コメントも正しく扱うため |
| `*.ini` (設定ファイル) | UTF-8 | configparser のデフォルト |
| `*.md` (ドキュメント) | UTF-8 | 標準 |

### 注意事項

- `.bas` ファイルを編集・新規作成するときは必ず **SHIFT-JIS (CP932)** で保存すること。
  UTF-8 で保存すると VBA エディタでのインポート時に日本語が文字化けする。
- `.py` ファイルは **UTF-8 (BOM なし)** で保存すること。
